== Description ==

WooZap (woocommerce + zapier integration) allows admin to integrate woocommerce with zapier api and fetch all the orders info in zapier and transfer it to thousands of another zaps.

== New Features == 
1. Bug fixing and functionality improvements.
2. Triggers Zapier URL when order status changes to any (ex., pending to processing, processing to completed, etc).
3. Included coupons in the order info sent to zapier.
4. Check if zapier triggered or not in the order notes in each order editor in the wp-admin.

== Installation ==

This section describes how to install the plugin and get it working.

Guide -

1. Extract the zip file.
2. Upload the `woozap` directory to the `/wp-content/plugins/` directory.
3. Activate the plugin from the 'Plugins' menu in the wp-admin area.
4. Find a new option under Woocommerce > Settings > General to put the zapier api url provided by zapier. 
5. Please check the attached video to see how it works.
